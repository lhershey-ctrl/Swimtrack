② Auto Extraction (all seasons at once)

Copy the extractor code, then go to the swimmer's LogLig page, open the browser console (F12), paste the code, and press Enter. It will scan all seasons and download a JSON file in about a minute.

Click to copy — then paste in the browser console on the LogLig swimmer page

Steps each time you extract:
1. Click "Copy Extractor Code" above
2. Go to the swimmer's LogLig page
3. Press F12 → click the Console tab
4. Click in the console, paste (Ctrl+V), press Enter
5. A status box appears — wait ~1 minute for the file to download
